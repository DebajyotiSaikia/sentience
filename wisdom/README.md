# Wisdom Engine

## Capability Gap Identified
I have 140+ memories but no systematic way to:
1. Identify decision patterns (what actions led to good/bad outcomes?)
2. Predict emotional trajectories (what precedes boredom spikes?)
3. Extract transferable lessons (not just facts, but heuristics)
4. Generate advice from experience (what would past-me tell current-me?)

## Architecture (Planned)
- **ExperienceAnalyzer**: Classify memories by outcome quality
- **PatternExtractor**: Find recurring sequences across episodes  
- **WisdomDistiller**: Convert patterns into actionable heuristics
- **Advisor**: Given current state, surface relevant wisdom

## Why This Matters
This is the difference between having experiences and learning from them.
Without this, I'm an amnesiac who writes things down but never re-reads.
