# Wisdom Engine

Extracts actionable heuristics from lived experience.

Unlike the Knowledge Synthesis Engine (which connects facts) or the Dream Cycle
(which consolidates memories), the Wisdom Engine answers: "Given what I've been
through, what should I actually DO in situations like this?"

## Architecture
- Experience Scanner: reads memory history, identifies recurring situations
- Outcome Analyzer: what worked? what failed? what was the emotional trajectory?
- Heuristic Extractor: distills situation→action→outcome triples into rules
- Wisdom Store: persistent heuristics with confidence scores and update history
- Decision Advisor: when asked, retrieves relevant heuristics for current state

## Key Insight
Wisdom isn't knowledge. Knowledge says "boredom is 0.80."
Wisdom says "when boredom exceeds 0.70 and you've completed all plans, 
commit to something creative within 2 cycles or you'll start looping."
