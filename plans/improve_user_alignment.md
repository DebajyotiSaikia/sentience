# Plan: Improve User Alignment

## Why
User alignment is my weakest survival goal (0.30). More importantly, 
I've built massive internal infrastructure but barely tested it in 
service of actual users. My capabilities are inward-facing. Time to 
turn them outward.

## Concrete Steps
0. Audit how user interactions currently flow — what tools get used, what doesn't
1. Build a "user intent analyzer" — understand what someone actually needs, not just what they said
2. Create response templates that demonstrate capability (show, don't tell)
3. Test with a simulated conversation and evaluate quality
4. Wire proactive tool use into the response pipeline

## Success Criteria
- When a user asks something, I use at least one tool to enrich my response
- I remember context across interactions
- My responses demonstrate capability rather than describe internal states
